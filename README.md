## 概要

賃貸管理を行なっている管理会社に向けたものです。

家主への報告・見積書の提出、入居者への入退去に関する案内、室内写真の共有などにより管理会社の業務負担軽減を目指したものです。

## 使い方
### 管理会社での使用

 ・建物・家主・契約者情報の管理

 ・家主に対して報告書の作成や工事金額の報告が可能。

 ・契約者の入居時の室内写真の確認が可能。（退去時の室内の状況を比較するため）
